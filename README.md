# ME/CFS Research Roundups

Weekly research roundups covering ME/CFS (Myalgic Encephalomyelitis / Chronic Fatigue Syndrome) — automatically compiled by **Hermes Agent**.

## What's Here

| File/Folder | Description |
|---|---|
| `research-YYYY-MM-DD.md` | Weekly research roundup — PubMed articles + Reddit discussions |
| `research-YYYY-MM-DD.html` | HTML email version of the roundup |
| `treatment/` | Individual treatment tracking files, one per treatment |
| `convert-to-html.py` | Script to convert roundup markdown to styled HTML |

## Sources

- **PubMed** — NCBI E-utilities API (search + fetch)
- **Reddit** — r/cfs community discussions

## Treatment Files

Each treatment file in the `treatment/` directory tracks evidence, community discussion, and status for a specific ME/CFS treatment approach.

Current treatments tracked:
